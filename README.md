# Shades

A remake of [Shades](https://play.google.com/store/apps/details?id=com.uovo.blocks.shades&hl=en) by [Uovo](http://www.uovo.dk) written in Java.

## Motivation

This game is made for Object Oriented Programming at NYU Shanghai.
